# delay
Delay up to the specified time

## Install
- Clone
- Add execute right:
  ```bash
  chmod a+x delay.py
  ```
- In UNIX-like system: copy to /usr/bin
  ```bash
  sudo copy delay.py /usr/bin
  ```
- In Windows: copy to any PATHs folder (not tested %) )

## Usage
```bash
delay.py 12:17 && ls
```
